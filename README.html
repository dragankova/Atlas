Atlas Team Member HTML Generator

Download zipped file

  1. Unzip files in the folder.

Hosted Use (GitHub Pages)

  1. Copy atlas_presentation_generator.html, Atlas_IUL_Template.html, and Atlas_Annuity_Template.html to GitHub.
  2. Create GitHub pages
  3. Open atlas_presentation_generator.html from the GitHub Pages URL.
  4. Select a template from the dropdown.
  5. Fill in your information. Ignore the Carriers input field.
  6. Select your photo.
  7. Click Preview.
  8. Click Download HTML.

Local Use

  1. Double-click start-server.bat.
  2. Your browser will open http://localhost:8000/generator.html.
  3. Select a template from the dropdown.
  4. Fill in your information. Ignore the Carriers input field.
  5. Select your photo.
  6. Click Preview.
  7. Click Download HTML.
Files that must stay together in the same folder:
generator.html
Atlas_Team_Template_IUL.html
Atlas_Team_Template_Anuity.html
start-server.bat
